Money Play. How should Red play 53?
pips Red=88 White=123
Red has doubled so must win through his play of the checkers. He can pay now with 13/10, 13/8 or pay later with 8/5, 6/1. Which is correct?

The answer is that the two plays have identical equities and so either play is correct. What this position provides is a very good reference for this type of problem so please remember it for future use.

Note that if the cube was in the centre Red should have doubled before rolling but if he had rolled this 53 with the cube in the centre then 13/10, 13/8 would be a double blunder. Never give your opponent the chance to win the game without making him pay for that opportunity. In other words, you must double first before paying now.
