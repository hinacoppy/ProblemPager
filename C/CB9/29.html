Match Play. Red leads 4-0 to 7. Should Red double? If doubled, should White take?
pips Red=158 White=191
(CorrectAnswer= 0/0, ErrorScore= 0)
For money this would be a massive pass but at this score White can take because Red doesn’t get full value from his gammons plus White’s redoubles, if he can turn the position around, are very powerful.

So should Red play on for the undoubled gammon? The answer is no, he should double now. There is huge value to getting to 6-1 Crawford in those variations where he wins a single game.

Not doubling is a blunder and don’t forget, a lot of Whites will be dropping as they incorrectly evaluate this position using money criteria in which case Red will neatly steal a point.
