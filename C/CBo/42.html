Match Play. Red leads 2-0 to 7. Should Red double? If doubled, should White take?
pips Red=106 White=133
Unlike the last couple of positions this only a double, not a redouble, so Red does not need to be so cautious with the cube and in fact this is a huge double.

Twenty-seven pips ahead and with an extra landing point in his outer board (the 9-pt) Red must double. For money, this would be a small pass. At this score, because of the redoubling potential, White has an easy take.

Not doubling is a blunder while a pass by White is merely a bad error.
