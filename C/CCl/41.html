It is nearly always better to make a point rather than slot it. Therefore, 7/5, 6/5 is better than 13/11, 6/5.

Is splitting the rear checkers with something like 24/22, 24/23 or 24/22, 6/5 any better? Quite simply, no. The 5-pt asset outweighs the other options although it is relatively close. The asset will last for the rest of the game so Red should make it now.
