I always do the money play first and then use that as a reference for the match play decision.

For money this looks very volatile with lots of good double hits for Red so he should certainly be doubling. Meanwhile White's board is as strong as Red's and there are plenty of counter chances so it looks like a reasonable take. A rollout agrees with this evaluation.

At the match score it's a very different story. A gammon for Red wins him the match with no wastage so this is a very efficient double which White has to drop.
