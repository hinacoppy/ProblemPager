Match Play. Red trails 4-5 to 7. Should Red double? If doubled, should White take?
pips Red=98 White=77
(CorrectAnswer= 0/0, ErrorScore= 0)
Obviously no double for money but getting to the Crawford Game is powerful and there are some gammons for Red here.

The possibility of future shots if this one is missed and the state of White's home board all add up to make this a double by a small margin. The trick is not in the analysis but in seeing the possibility in the first place.
