The match score changes things because if Red wins a 4-cube he will reach the Crawford Game which is very powerful. Red has priming and blitzing possibilities and the position is very volatile so Red must double now. Not redoubling is a bad blunder, as you can see from the rollout.

White still has a very comfortable take although you will get a few drops here because Red’s position looks more threatening than it is.
