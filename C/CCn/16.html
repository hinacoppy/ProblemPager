Interestingly XG’s initial evaluation’s of this position that it is a clear double for Red but once it looks deeper (via a rollout) it completely reverses its original decision.

The problem is that Red loses as many gammons as White and a gammon loss is the end of the match.

Giving White perfect cube efficiency is not a good idea and so Red should not double. Doubling is a bad error but not a blunder.
