Match Play. 4-4 to 7. Should Red double? If doubled, should White take?
pips Red=124 White=139
Yet another position where there is a huge difference between the cube action for money and at this actual score.

For money this is a borderline no double but not doubling at 3-away, 3-away is a big blunder. At this score you should both double earlier than normal and drop earlier than normal. This is a great double at this score with the volatility sky-high and a gammon wins the match for Red.

White has enough structure plus racing chances to make it a comfortable take.
