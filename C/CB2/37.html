Money Play. How should Red play 42?
pips Red=61 White=83
(CorrectAnswer= 0/0, ErrorScore= 0)
This is an example of what Paul Magriel calls “duplication for the sake of duplication”.

Red can play 16/10 duplicating fours but leaving sixteen shots which is a lot against White's five-point board. Red's board is already busted so weakening it some more with 5/3, 5/1 doesn't really matter.

Meanwhile White's position is likely to weaken next turn. All this implies that Red should temporise with 5/3, 5/1 although it is relatively close. ( 0.02 equity points)

Note that if either player owned the cube then 16/10 becomes a blunder because now gammons are active and White will be playing on after a hit and fan sequence.
