If this was the 1970s then 21/20, 21/18 would probably have been played but this is an area of the game where the bots have taught us a great deal.

Red wants to attack and close out the White checker on his 4-pt so 13/10 is clear. What perhaps is not so clear is that the correct one is 21/20, exposing five blots. However, the key point is that any 4 by White, with the exception of 44, will leave multiple return shots if White elects to hit.

In other words the blots protect each other, a technique unknown before the bots but now relatively common knowledge.
