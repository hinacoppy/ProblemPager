This is the continuation of Problem 102. Should Red make the 2-pt or run off White’s bar-point?
The problem with making the 2-pt is that when White enters with a 4 Red will find it difficult to run home.
The priority is getting the checkers home and 18/14, 18/13 is correct. It will win slightly fewer gammons but increases the game winning percentage by 6%. That is a huge margin and 7/2, 6/2 is actually a blunder­­­­.
Even at 4-away vs 2-away running off the bar is correct. First win the game and then win the gammon - a useful maxim.
