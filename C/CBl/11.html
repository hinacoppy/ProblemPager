The difference between the various plays will be small but small differences add up over time.

There has long been discussion/debate on which point to stack your spare checkers in positions like this so I decided to run some analyses. The answer that came back was that the 3-pt is where to make your stack and so the correct play in this position is 6/3, 5/3. 6/4, 5/2 is not quite an error but only by a very small margin.
