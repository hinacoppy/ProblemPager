Money Play. Should Red double? If doubled, should White take?
pips Red=147 White=163
I thought it was time for a reference position. Positions of this genre can be found in Kit Woolsey's “The Backgammon Encyclopedia” Chapter 1.

This one is no double (only just) and easy take. If White's blot on Red's 2-pt were on Red's 4-pt it would be double/take because the blot is easier to attack there and when the point is made it is a better point than the 2-pt.

Anyone not familiar with these early blitz reference positions should get a copy of Woolsey's book - it is gold dust.
